# Shreya's Developer Portfolio

Welcome. This is my personal developer portfolio built with React and customized on top of the DeveloperFolio template. It showcases who I am, what I build, and how to reach me.

The design uses smooth animations, and a clean layout focused on readability.

---
### 🔗 Live Link
https://shreyachaudhary1112.github.io/Portfolio/


## ✨ Features

- Hero section with a short introduction and resume link  
- Skills with animated proficiency bars  
- Education and Work Experience timeline  
- Projects section highlighting selected work  
- Achievements and certifications  
---

## 🧑‍💻 About this project

- Framework: React (Create React App)  
- Styling: SCSS modules with a global color system  
- Animations: Lottie animations and subtle transitions  
- Integration: GitHub API for pinned repositories and profile data (optional)  

Most of the content is configured through a single file, so I can update my information without touching the layout or logic.
